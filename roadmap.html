3. Third Module <2022-01-19 Wed>

  1. The main file for this class will be teach/theory-day.html.

  2. Things to read before class. There are three short opinion pieces here. Give them a skim with the goal of priming your thinking for whether you agree or disagree that there is a problem with theories in psychology and neuroscience.

    This is optional. At Companion and Optional Readings you will find links to two recent articles that tackle the idea of formal models and theory in psychology (much more so than neuroscience). I think these are great to read, but as I want you to read Suppes (see below). I am making these optional. If you have the time, great. If not, try to get to them later.

    This is important since I plan to base a lot of my discussion around it. Suppes, 1967, What is a scientific theory.

  3. Before class next week submit the github repo dropbox item.
